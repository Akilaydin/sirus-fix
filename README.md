# Исправление ошибок лаунчера Sirus

Небольшой `.bat`-скрипт для запуска `Sirus Launcher` в Windows без влияния глобальных переменных окружения proxy.

## Что делает скрипт

Скрипт перед запуском `Sirus Launcher` очищает переменные:

* `http_proxy`
* `https_proxy`
* `HTTP_PROXY`
* `HTTPS_PROXY`
* `NO_PROXY`
* `no_proxy`

После этого запускается `Sirus Launcher` уже без proxy-окружения.

## Как использовать

1. Сохранить скрипт в файл `run-sirus-without-proxy.bat`
2. Установить sirus launcher как обычно
3. Убедиться, что `Sirus Launcher` установлен по пути:
   `%LocalAppData%\Programs\sirus-open-launcher\Sirus Launcher.exe`
4. Запускать лаунчер через этот `run-sirus-without-proxy.bat`

### Если путь отличается

Если лаунчер установлен в другое место, нужно заменить строку запуска на свой путь.

## Зачем это нужно

У некоторых пользователей `Sirus Launcher` не может скачать патчи или саму игру с такими ошибками:
* `Ошибка подключения к серверам обновлений. Все серверы проверены. Следующая попытка через 1 минуту.`
* `Не удалось загрузить список патчей`
* `Информация об игровых серверах недоступна.`

## Другие возможные фиксы:

* Закрыть игру и лаунчер полностью, включая значок в системном трее, затем запустить снова.
* Запустить лаунчер от имени администратора.
* В настройках лаунчера сбросить или сменить сервер обновлений, например между `s-patches.ru` и `s-patches.pro`.
* В экспертных настройках проверить, что список патчей выбран `LIVE`. 
* Снять атрибут `Только чтение` с папки игры и всех файлов внутри неё. 
* Удалить `Client.dll` и `run.exe` из папки игры, после чего снова запустить лаунчер, чтобы он скачал их заново. 
* Добавить папки лаунчера в исключения антивируса.
* Переключиться на другого провайдера(например, раздать интернет с телефона)
* Создать нового windows пользователя и попробовать зайти с него
* Попробовать другие осуждаемые инструменты
